# K2P刷机指南

## 准备工作

1. **备份原厂固件**
   - 使用Breed或其他bootloader备份
   - 备份ART分区（无线校准数据）

2. **下载工具**
   - 网线
   - 电脑
   - 固件文件

## 刷机步骤

### 方法一：原厂固件升级
1. 连接路由器，访问管理界面（通常192.168.2.1）
2. 找到"固件升级"页面
3. 选择编译好的 \`.bin\` 文件
4. 开始升级，等待自动重启

### 方法二：使用Breed
1. 进入Breed恢复模式（按住Reset键通电）
2. 访问192.168.1.1
3. 选择固件更新
4. 上传固件文件
5. 点击更新

## 首次设置

1. **连接网络**
   - 刷机后路由器IP变为192.168.1.1
   - 电脑设置为自动获取IP

2. **登录路由器**
   \`\`\`bash
   ssh root@192.168.1.1
   # 首次登录无密码，立即设置密码
   passwd
   \`\`\`

3. **基本配置**
   \`\`\`bash
   # 设置时区
   uci set system.@system[0].timezone='CST-8'
   uci set system.@system[0].zonename='Asia/Shanghai'
   uci commit system
   
   # 重启服务
   /etc/init.d/system restart
   \`\`\`

## 故障排除

### 无法登录
- 检查IP地址是否正确
- 尝试使用网线连接
- 重置路由器（按住Reset键10秒）

### 无线信号弱
- 检查是否备份并恢复了ART分区
- 尝试不同的无线信道

### 网络不稳定
- 检查防火墙设置
- 更新驱动或固件
